IOTOPE Node Concept. Node administration and troubleshooting page. This application is a Proof of Concept, please threat it as is.
Node group: PUBLIC A node can be part of a group, that can limit or extend the applications allowed on this node.
Node name: UNNAMED Friendly name of the node within the group.
Learn Mode When learn mode is enabled, it will suppress all action associated (default or explicit) with the target.
Read Content. When reading content is enabled the node will read all the content from the target. This will enable richer information, but is slower.
Server: NOT IMPLEMENTED The node can connected to the public IOTOPE server or be part of a local private server.
Default application: NOT IMPLEMENTED A group can inf.
Cache Content. Caching information will enable faster reads, but should not be enabled when the content of a target can change.
Execute associated applications. Execute the associated applications with specific targets. If this is disabled, the default application will be executed.
Node token: FFFFFFFF Token that has been generated for this node.
